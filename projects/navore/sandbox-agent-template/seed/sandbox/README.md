# sandbox/

Where Navore work side projects live. Everything here is fair game to build, break, and rebuild
— that's the point. One subfolder per project:

```
sandbox/
├── site-replica/     ← a local replica of the real Navore site, for learning/prototyping
└── <your-project>/   ← whatever you're tinkering with
```

Build freely here. Just remember: this all commits to `Navore-Ops` and **only** `Navore-Ops`.
